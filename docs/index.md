KB1 is a simple keelboat design created to demonstrate the [Shipshape](https://github.com/shipshape-marine/shipshape) parametric CAD library and the design setup of the [Shipshape](https://github.com/shipshape-marine) organization. The entire boat---hull, deck, keel, rudder, rig---is generated from a single set of parameters, with all geometry, analysis, and documentation produced automatically.
